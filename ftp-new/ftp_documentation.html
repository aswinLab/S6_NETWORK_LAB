FTP Documentation

A line-by-line breakdown of the lightweight C-based FTP Server and Client implementation.

Server Core Logic (server.c)

void send_msg(int fd, const char *msg)

Utility wrapper to safely send a null-terminated string over a socket descriptor.
void send_msg(int fd, const char *msg) {
Defines the function. `fd` is the client socket, `msg` is the string to send.
send(fd, msg, strlen(msg), 0);
Uses the `send` syscall. `strlen(msg)` calculates the exact bytes to transmit, excluding the null terminator.
}
Ends the function.

int recv_line(int fd, char *buf, int size)

Reads characters one-by-one from the socket until a newline ('\n') is encountered.
int i = 0; char c;
Initializes an index counter `i` and a temporary character buffer `c`.
while (i < size - 1) {
Loops until the buffer is nearly full, leaving exactly 1 byte for the null terminator `\0`.
int n = recv(fd, &c, 1, 0);
Receives exactly 1 byte from the network socket into the variable `c`.
if (n <= 0) return n;
If `recv` returns 0 (client disconnected) or -1 (error), abort and return the error code immediately.
if (c == '\n') break;
If the received character is a newline, the full line has been read. Break out of the loop.
buf[i++] = c;
Store the character in the buffer and increment the index.
}
End of the reading loop.
buf[i] = '\0';
Append a null terminator to make the buffer a valid C string.
return i;
Returns the total number of bytes read (excluding the newline and null terminator).

const char *safe_name(const char *name)

Security function to prevent Path Traversal attacks (e.g., stopping clients from requesting `../../etc/passwd`). Extracts only the filename.
const char *p = strrchr(name, '/');
Finds the last occurrence of a forward slash ('/') in the provided path.
return p ? p + 1 : name;
If a slash was found, return the string starting immediately after it. Otherwise, the whole string is safe.

void do_list(int client)

Handles the `LIST` command. Reads the `storage` directory and sends the list of files to the client.
DIR *dir = opendir(STORE);
Opens the "storage" directory to read its contents.
if (!dir) { send_msg(client, "ERR\n"); return; }
If the directory fails to open, send an "ERR" message and abort.
char out[4096] = "";
Allocates a 4KB buffer to accumulate the names of the files.
while ((e = readdir(dir)))
Iterate through every entry in the directory.
if (e->d_name[0] != '.')
Ignore hidden files, the current directory (`.`), and the parent directory (`..`).
{ strcat(out, e->d_name); strcat(out, "\n"); }
Append the valid filename and a newline to our `out` buffer.
char hdr[64]; sprintf(hdr, "LIST %ld\n", (long)strlen(out));
Format the protocol header informing the client how many total bytes the file list will be.
send_msg(client, hdr);
Send the `LIST ` header.
send(client, out, strlen(out), 0);
Send the actual raw text block containing all the file names.

void do_get(int client, const char *name)

Handles the `GET` command. Streams a requested file from the server down to the client.
name = safe_name(name);
Sanitize the requested filename to prevent path traversal.
sprintf(path, "%s/%s", STORE, name);
Construct the full server-side path, e.g., "storage/myfile.txt".
if (stat(path, &st) < 0) { send_msg(client, "ERR\n"); return; }
Check if the file exists and grab its metadata. Send `ERR` if it fails.
sprintf(hdr, "SIZE %ld\n", (long)st.st_size);
Format the protocol header specifying the exact filesize to expect.
send_msg(client, hdr);
Send the `SIZE` header so the client knows how much to download.
int fd = open(path, O_RDONLY);
Open the file in Read-Only mode.
while ((n = read(fd, buf, BUF)) > 0)
Read the file into the 1024-byte buffer iteratively until EOF is reached.
send(client, buf, n, 0);
Send the exact chunk of bytes just read over the socket to the client.

void do_put(int client, const char *name, long size)

Handles the `PUT` command. Receives an uploaded file from the client and saves it to disk.
int fd = open(path, O_WRONLY | O_CREAT | O_TRUNC, 0644);
Opens/Creates a file in Write-Only mode. `O_TRUNC` empties it if it exists. Sets permissions to 644.
send_msg(client, "READY\n");
Sends a handshake `READY` message telling the client to start beaming the raw bytes.
long rem = size;
Initializes a tracker for how many bytes are left to download.
while (rem > 0) {
Loops until all expected bytes have been received.
int chunk = rem > BUF ? BUF : rem;
Calculates how much to read: max 1024 (BUF) or exactly whatever is remaining. Prevents reading past EOF into the next command.
int n = recv(client, buf, chunk, 0);
Receive the calculated chunk from the network.
write(fd, buf, n);
Write those exact bytes directly into the file on disk.
rem -= n;
Subtract the amount written from the remaining tracker.
}
End of upload loop.
send_msg(client, "OK\n");
Send `OK` to let the client know the upload succeeded.

void handle_client(int client)

The primary command loop for an active client connection. Parses text and routes to actions.
while (recv_line(client, line, BUF) > 0) {
Continuously listens and reads one full string command (until '\n') from the client.
sscanf(line, "%s %s %ld", cmd, arg, &fsize);
Parses the string into 3 parts: Command (`cmd`), Argument/Filename (`arg`), and Filesize (`fsize`).
if (!strcmp(cmd, "PUT")) do_put(client, arg, fsize);
Routes standard string commands like "PUT", "GET", "LIST" to their respective functions.
else if (!strcmp(cmd, "QUIT")) break;
If the client requests to QUIT, the loop breaks, closing the connection.

Client Core Logic (client.c)

int recv_all(int fd, char *buf, long len)

Helper function to guarantee that exactly `len` bytes are received from the network. TCP doesn't guarantee full delivery in one go.
long got = 0;
Counter for bytes successfully received so far.
while (got < len) {
Keep trying until we have collected the target amount of bytes.
int n = recv(fd, buf + got, len - got, 0);
Read whatever is available on the socket directly into the buffer, offset by `got` so we don't overwrite earlier data. We ask for exactly what remains: `len - got`.
if (n <= 0) return -1;
Connection died or errored out mid-transfer. Abort.
got += n;
Increase the success counter by the chunk we just got.
}
End of loop.

void do_list(int sock)

Client implementation of LIST. Requests the directory state and safely prints it.
send(sock, "LIST\n", 5, 0);
Sends the textual command to trigger the server's LIST logic.
recv_line(sock, buf, BUF);
Reads the first response line, expecting the `LIST ` format.
sscanf(buf, "LIST %ld", &size);
Extracts the total byte size of the directory string the server is about to send.
char *data = malloc(size + 1);
Dynamically allocates memory precisely matched to the incoming payload (plus null terminator).
recv_all(sock, data, size);
Downloads the entire payload block in one guaranteed operation using our helper.
data[size] = '\0'; printf("%s", data);
Terminates the string and prints the file list cleanly to the user's terminal.
free(data);
Frees the dynamically allocated memory to prevent memory leaks.

Client entry point. Connects to server and provides an interactive terminal prompt.
while (1) {
Infinite loop to keep the interactive CLI running.
printf("ftp> ");
Prints the classic FTP prompt.
if (!fgets(input, BUF, stdin)) break;
Reads user input from the keyboard (stdin). Breaks out if EOF (Ctrl+D) is detected.
input[strcspn(input, "\n")] = '\0';
Strips the trailing newline character inserted by `fgets`.
sscanf(input, "%s %s", cmd, arg);
Parses out the base command (like GET) and its argument (like filename.txt).
if (!strcmp(cmd, "PUT")) do_put(sock, arg);
Routes client-side execution to handle local file reading, chunking, and sending.
